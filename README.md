### FEATURED PROJECT

> **[klartext-tools.com](https://klartext-tools.com/en/) — Free Online Tools, Calculators & Guides**
>
> A growing collection of 60+ free, privacy-friendly tools that run entirely in the browser. No accounts, no tracking, no paywalls.
>
> Built with **Eleventy (11ty)** · Vanilla JS · Nunjucks · Multilingual (EN/DE) · Static, CDN-deployed

| Category | Highlights |
|---|---|
| 🛠 Developer Tools | JSON formatter, Regex tester/generator/explainer, JWT decoder, Cron builder, Base64/Hashing |
| 🎮 Gaming Tools | FPS calculator, Sensitivity converters (CS2, VALORANT, Fortnite), Aim visualizer |
| 📐 Math & Science | Function plotter, Statistics calculator, Unit converter, Equation solver |
| 🌐 Web Utilities | SEO meta tag generator, Sitemap validator, Open Graph preview, UTM builder |
| 🏠 Handyman | Paint, tile, flooring & concrete calculators |
| 🖥 Diagnostics | Keyboard tester, Typing speed test, Reaction time test |
